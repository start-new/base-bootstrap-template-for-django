# Base Bootstrap Template For Django

This cookiecutter template creates a basic Bootstrap base_html for Django.

## Dependence

This is a template tailored for the cookiecutter project templating tool. To use it, you first need to install the Python 3 programming langage and the cookiecutter utility using your prefered python package manager.

## How to user base-bootstrap-template-for-django

Once cookiecutter installed on your computer, creating un root templates folder in your django project can be done as follows:

```
$ cookiecutter gh:start-new/base-bootstrap-template-for-django
```

Cookiecutter will ask questions during the creation process to customize the template to your needs.
